جریان کنترل: شرطی‌ها در پایتون

تصمیم‌گیری در برنامه‌های خود با استفاده از if، elif و else.

جریان کنترل چیست؟

جریان کنترل به ترتیبی اشاره دارد که دستورات در یک برنامه کامپیوتری اجرا می‌شوند. در پایتون، دستورات جریان کنترل به شما امکان می‌دهند که برنامه‌های خود را پویا و هوشمند کنید، به طوری که بر اساس شرایط مختلف، مسیرهای متفاوتی را طی کنند. دو دسته اصلی از دستورات جریان کنترل وجود دارد: شرطی‌ها و حلقه‌ها.

این درس بر دستورات شرطی تمرکز دارد. برای یادگیری در مورد حلقه‌ها، می‌توانید به صفحه جریان کنترل: حلقه‌ها مراجعه کنید.

دستور if

دستور if برای اجرای یک بلوک کد تنها در صورتی که یک شرط خاص True باشد، استفاده می‌شود. اگر شرط False باشد، بلوک کد if نادیده گرفته می‌شود.

ساختار دستور if:


if شرط:
    # بلوک کدی که اگر شرط True باشد اجرا می‌شود

مثال:


سن = 18
if سن >= 18:
    print("شما مجاز به رای دادن هستید.")

خروجی:


شما مجاز به رای دادن هستید.

دستور else

دستور else اغلب با if استفاده می‌شود. این دستور یک بلوک کد را زمانی اجرا می‌کند که شرط if اولیه False باشد.

ساختار دستور if...else:


if شرط:
    # بلوک کدی که اگر شرط True باشد اجرا می‌شود
else:
    # بلوک کدی که اگر شرط False باشد اجرا می‌شود

مثال:


نمره = 75
if نمره >= 60:
    print("شما قبول شدید!")
else:
    print("متاسفانه، شما مردود شدید.")

خروجی:


شما قبول شدید!

دستور elif (Else If)

دستور elif (کوتاه شده "else if") به شما امکان می‌دهد چندین شرط را بررسی کنید. اگر شرط if اولیه False باشد، پایتون شرط elif را بررسی می‌کند و این روند تا زمانی که یک شرط True پیدا شود ادامه می‌یابد.

ساختار دستور if...elif...else:


if شرط1:
    # بلوک کد برای شرط1
elif شرط2:
    # بلوک کد برای شرط2
elif شرط3:
    # بلوک کد برای شرط3
else:
    # بلوک کد اگر هیچ یک از شرایط بالا True نباشد

مثال: سیستم درجه‌بندی


درجه = 85

if درجه >= 90:
    print("درجه شما: الف")
elif درجه >= 80:
    print("درجه شما: ب")
elif درجه >= 70:
    print("درجه شما: ج")
elif درجه >= 60:
    print("درجه شما: د")
else:
    print("درجه شما: و")

خروجی:


درجه شما: ب

عملگرهای مقایسه‌ای

شرط‌ها در دستورات if، elif و else از عملگرهای مقایسه‌ای برای ارزیابی روابط بین مقادیر استفاده می‌کنند:

  • ==: برابر با
  • !=: نابرابر با
  • >: بزرگتر از
  • <: کوچکتر از
  • >=: بزرگتر یا مساوی با
  • <=: کوچکتر یا مساوی با

عملگرهای منطقی (Logical Operators)

برای ترکیب چندین شرط، می‌توانید از عملگرهای منطقی استفاده کنید:

  • and: اگر هر دو شرط True باشند، نتیجه True است.
  • or: اگر حداقل یکی از شرط‌ها True باشد، نتیجه True است.
  • not: نتیجه شرط را برعکس می‌کند (True را False و False را True می‌کند).

مثال:


سن = 25
کشور = "ایران"

if سن >= 18 and کشور == "ایران":
    print("مجاز به رانندگی در ایران هستید.")

has_ticket = True
is_vip = False

if has_ticket or is_vip:
    print("شما می‌توانید وارد شوید.")

is_raining = True
if not is_raining:
    print("امروز هوا آفتابی است.")
else:
    print("باران می‌بارد.")

با درک دستورات شرطی، شما توانایی ایجاد برنامه‌هایی را خواهید داشت که می‌توانند بر اساس ورودی‌ها یا شرایط مختلف تصمیم‌گیری کرده و به صورت هوشمندانه عمل کنند.
