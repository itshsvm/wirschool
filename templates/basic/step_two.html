انواع داده‌ها و متغیرها در پایتون

درک بلوک‌های ساختاری داده در پایتون و نحوه کار با آن‌ها.

متغیرها در پایتون

متغیرها در برنامه‌نویسی مانند ظرف‌هایی هستند که برای ذخیره مقادیر داده استفاده می‌شوند. در پایتون، نیازی به اعلام نوع متغیر قبل از استفاده از آن نیست؛ پایتون به صورت خودکار نوع را بر اساس مقداری که به آن اختصاص می‌دهید، تشخیص می‌دهد. متغیرها را می‌توان پس از اختصاص مقدار، تغییر داد.

نام‌گذاری متغیرها

  • نام متغیر باید با یک حرف یا علامت زیرخط (underscore) شروع شود.
  • نمی‌تواند با یک عدد شروع شود.
  • فقط می‌تواند شامل حروف الفبا-عددی و زیرخط (A-z, 0-9, and _) باشد.
  • به حروف کوچک و بزرگ حساس است (age، Age و AGE سه متغیر متفاوت هستند).

مثال:


نام = "علی"
سن = 30
قیمت = 99.99
is_active = True

انواع داده‌ها در پایتون

انواع داده‌ها، دسته‌بندی‌هایی هستند که مقادیری را که یک متغیر می‌تواند نگه دارد، تعیین می‌کنند. پایتون دارای انواع داده‌های داخلی مختلفی است که در ادامه به برخی از رایج‌ترین آن‌ها می‌پردازیم:

1. اعداد (Numbers)

پایتون سه نوع عددی را پشتیبانی می‌کند:

  • اعداد صحیح (int): اعداد کامل، مثبت یا منفی، بدون اعشار. می‌توانند به هر طولی باشند.
  • اعداد اعشاری (float): اعداد اعشاری، مثبت یا منفی، شامل یک یا چند رقم اعشار.
  • اعداد مختلط (complex): اعداد به فرم a + bj که a قسمت حقیقی و bj قسمت موهومی است.

مثال:


age = 25              # int
pi_value = 3.14       # float
complex_num = 1 + 2j  # complex

2. رشته‌ها (Strings)

رشته‌ها در پایتون دنباله‌ای از کاراکترها هستند که با کوتیشن تکی (') یا دوتایی (") احاطه شده‌اند. می‌توانید از کوتیشن سه‌گانه (""" یا ''') برای رشته‌های چندخطی استفاده کنید.

مثال:


greeting = "سلام، دنیا!"
name = 'مریم'
multi_line_string = """این یک رشته
چندخطی است."""

3. بولین‌ها (Booleans)

نوع داده بولین تنها دو مقدار ممکن دارد: True (درست) یا False (غلط). این مقادیر اغلب در عبارات شرطی و حلقه‌ها برای تصمیم‌گیری استفاده می‌شوند.

مثال:


is_student = True
has_license = False

4. لیست‌ها (Lists)

لیست‌ها مجموعه‌های مرتب و قابل تغییر از آیتم‌ها هستند. آیتم‌ها می‌توانند از انواع داده‌های مختلفی باشند و با کروشه ([]) تعریف می‌شوند.

مثال:


fruits = ["سیب", "موز", "گیلاس"]
numbers = [1, 2, 3, 4, 5]
mixed_list = ["سلام", 123, True]

5. تاپل‌ها (Tuples)

تاپل‌ها مانند لیست‌ها هستند، اما غیرقابل تغییر (immutable) هستند. به این معنی که پس از ایجاد، نمی‌توانید آیتم‌های آن‌ها را تغییر دهید، اضافه کنید یا حذف کنید. با پرانتز (()) تعریف می‌شوند.

مثال:


coordinates = (10.0, 20.0)
colors = ("قرمز", "سبز", "آبی")

6. مجموعه‌ها (Sets)

مجموعه‌ها، مجموعه‌های نامرتب و بدون تکرار از آیتم‌ها هستند. با آکولاد ({}) تعریف می‌شوند، اما برای تعریف یک مجموعه خالی باید از set() استفاده کنید (زیرا {} یک دیکشنری خالی ایجاد می‌کند).

مثال:


unique_numbers = {1, 2, 3, 2, 4} # {1, 2, 3, 4}
vowels = {"a", "e", "i", "o", "u"}

7. دیکشنری‌ها (Dictionaries)

دیکشنری‌ها مجموعه‌های نامرتب از جفت‌های کلید-مقدار هستند. هر کلید باید منحصر به فرد باشد. با آکولاد ({}) تعریف می‌شوند.

مثال:


person = {
    "نام": "احمد",
    "سن": 28,
    "شهر": "تهران"
}

با درک این انواع داده‌های اساسی و نحوه استفاده از متغیرها، شما آماده‌اید تا منطق پیچیده‌تری را در برنامه‌های پایتون خود بسازید.
